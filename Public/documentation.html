Documentation page

Objective

This is the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

End-Point Documentation

URL
Returns list of all movies /movies GET JSON object holding the information on the movies
Returns data about a single movie in the list by title /movies/:title GET :title Json object holding the information on a single movie
Returns data about a genre of moive by title /movies/:genre/:name GET :genreName Json object holding the information on a genre of movies
Returns data about a director by name /movies/:directors/:name GET :directorName Json object holding information on a director of movies by name
Allows users to add a movie to the data base /movies POST A JSON object holding information on a movie to be added to list in a form like: { "title":"string", "description":"string"} A JSON object holding information on a movie that was added to the list in a form like: { "title":"string", "description":"string"}
Allows new users to register to the site /users POST A JSON object holding information on a user to be added to list in a form like: { "username":"string", "password":"string", "email":"string", "date of birth":"string"} A JSON object holding information on a user that was added to the list in a form like: { "username":"string", "password":"string", "email":"string", "date of birth":"string"}
Allows a user to update their username /user/:Username PUT :Username Json object holding information on an updated username
Allows a user to add movies to their favorites /user/:Username/:movies/:title: POST :Username Json object holding information on an updated favorite movies list
Allows a user to remove movies from their favorites /user/:Username/:movies/:title: DELETE :Username Json object holding information on an updated favorite movies list
Allows a user to deregister from the site /user/:Username DELETE :Username Text letting user know they have been deregistered
